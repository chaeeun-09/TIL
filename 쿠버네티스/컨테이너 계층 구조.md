# 컨테이너의 계층 구조

![alt text](image.png)
컨테이너 계층 구조는 아래에서 위로 갈 수록 추상화 수준이 점점 높아진다.

- **낮은 추상화 (구체적, 하드웨어 장치 속함)**
    
    Layer 1 : 물리적 장비 
    
    Layer 2 : 가상화 - 물리 서버를 논리적으로 나눔
    
    Layer 3 : OS - 하드웨어를 추상화 해서 프로그램 실행 환경 제공
    
- **중간 추상화**
    
    Layer 4  : OS를 추상화 해서 격리된 실행 환경 제공
    
- **높은 추상화 (추상적,비지니스 로직에 가까움)**
    
    Layer 5 : 오케스트레이션
    
    Layer 6 : 개발자는 코드만 작성 , 나머지는 플렛폼에서 자동적으로 처리
